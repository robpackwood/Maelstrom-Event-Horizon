# Asset Provenance Report

Audit date: 2026-07-23

Product: Maelstrom - Event Horizon 1.0.0

Publisher: mroblivious

Store product ID: 9NW0R8TS1F5D

Package identity: mroblivious.MaelstromEventHorizon

## Scope

The audit covers all image, icon, MP3, and WAV files embedded in the game
project, plus all Microsoft Store package images. The canonical per-file list
is `asset-manifest.csv`.

At audit time the game project contained:

- 72 PNG files, 1 ICO file, 12 MP3 files, and 1 WAV file.
- 6 additional PNG files used by the Microsoft Store package.
- Runtime sound effects synthesized by project code in
  `Infrastructure/Audio/SynthAudio.cs` and
  `Infrastructure/Audio/SynthSoundEffectLibrary.cs`.

## Evidence standard

The following labels are used in the manifest:

- `VERIFIED_EXACT`: the local file and source download have the same SHA-256,
  and the source page identifies the author and license.
- `LICENSE_VERIFIED_TRANSCODE`: the source page identifies the author and
  license, and the local filename/use maps to that work, but the shipped MP3 is
  a transcode of an OGG source and no conversion log was retained.
- `PROJECT_GENERATED_DERIVATIVE_UNCLEARED_SOURCE`: generated from a local source
  whose own provenance is unresolved.
- `DECLARATION_REQUIRED`: a file is hashed and present in Git history, but
  ownership or license evidence is absent.

## Artwork findings

No license file, source URL, layered source file, image-generation prompt,
generation-service terms snapshot, purchase receipt, work-for-hire agreement,
or signed author declaration was found for the artwork.

Git history records that the image files were added in commits
`1ba4b79849f9de04f4a182dd0a2185894925dec6` and
`c9c70ff03fe48a1129cace54bee1cb2c436785e1`, authored by Rob Packwood. This is
useful custody evidence, but it does not by itself prove copyright ownership.

The six Store tile and splash images are generated by
`MaelstromEventHorizon.Package/Generate-StoreAssets.ps1` from
`Assets/maelstrom-icon.png`. They inherit the unresolved provenance of that
source icon.

## Title findings

The title `Maelstrom - Event Horizon` appears in assembly metadata, the window
title, package metadata, and Store reservation. The repository contains no
trademark clearance search, registration, assignment, license, or legal
opinion for the title.

Microsoft Store name reservation is not a trademark clearance. Microsoft Store
Policy 10.1.1 requires a unique, non-misleading product title and prohibits use
of another product's name or metadata unless it is also published by the same
provider. Policy 11.2 requires all content, including names, images, and sounds,
to be original, licensed, or otherwise legally permitted.

Official references:

- Microsoft Store Policies:
  https://learn.microsoft.com/en-us/windows/apps/publish/store-policies
- USPTO federal trademark searching:
  https://www.uspto.gov/trademarks/search/federal-trademark-searching
- WIPO Global Brand Database:
  https://www.wipo.int/en/web/global-brand-database

The title remains `NOT CLEARED` until an appropriate clearance review is
completed for every market in which the game will be offered.

## Audit limitations

This audit verifies repository contents, public license-page statements, and
selected exact file matches. It cannot establish that an uploader owned every
right they purported to license, that a trademark is available, or that no
unregistered/common-law rights exist. Creative Commons licenses also disclaim
warranties and do not necessarily cover trademark, publicity, privacy, or other
rights.
