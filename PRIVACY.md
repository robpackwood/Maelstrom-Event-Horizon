# Privacy Policy for Maelstrom: Event Horizon

Effective date: July 23, 2026

Maelstrom: Event Horizon is published by mroblivious. This policy explains how
the game handles information when it is installed and played on Windows.

## Summary

Maelstrom: Event Horizon is an offline, single-player game. The game does not
include advertising, analytics, telemetry, user accounts, or online services.
It does not transmit personal information or gameplay data to the publisher or
to third parties.

## Information handled by the game

The game stores the following information locally on the Windows device:

- A pilot name or call sign entered by the player after earning a qualifying
  high score.
- High-score information associated with that name or call sign, including the
  score, wave reached, and date and time achieved.
- Control bindings selected by the player.
- Display and audio preferences, including full-screen mode and volume levels.

Players are encouraged to enter a call sign or nickname rather than a real
name. The game does not request an email address, postal address, phone number,
payment information, precise location, contacts, photographs, microphone or
camera data, or government identifiers.

## How information is used

Locally stored information is used only to:

- Display the top-ten high-score table on that device.
- Restore the player's selected controls and display and audio preferences.

The game does not use this information for advertising, profiling, tracking,
marketing, or automated decision-making.

## Storage, security, and retention

The game writes its local files to:

`%LOCALAPPDATA%\MaelstromEventHorizon`

These files remain on the device until they are deleted by the user. They are
stored within the current Windows user's profile and are protected by the
access controls provided by Windows. The game does not upload or remotely back
up these files.

For compatibility with earlier versions, the game may copy existing settings
or scores from:

`%LOCALAPPDATA%\MaelstromDegenerateGreed`

## Sharing and disclosure

The game does not transmit locally stored information and does not share it
with the publisher, advertisers, analytics providers, or other third parties.

Microsoft may independently process information when the game is acquired,
downloaded, installed, updated, or reviewed through the Microsoft Store. That
processing is performed by Microsoft outside the game and is governed by the
[Microsoft Privacy Statement](https://privacy.microsoft.com/privacystatement).

## Player choices and access

Players may inspect their locally stored information by opening the JSON files
in `%LOCALAPPDATA%\MaelstromEventHorizon`.

To erase the game's locally stored scores and preferences:

1. Close Maelstrom: Event Horizon.
2. Delete the `%LOCALAPPDATA%\MaelstromEventHorizon` folder.
3. If data from an older version exists, also delete the
   `%LOCALAPPDATA%\MaelstromDegenerateGreed` folder.

Because the game does not receive or transmit this information, the publisher
does not have a remote copy to access, correct, or delete.

## Children's privacy

The game is not designed to request personal information from children and
does not transmit player information off the device. A child should use a
fictional call sign rather than a real name on the local high-score table.

## Changes to this policy

This policy may be updated if the game's features or information practices
